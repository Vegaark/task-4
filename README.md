## Objective
Build a REST API to manage user data using Flask with GET, POST, PUT, DELETE routes.

## Tools Used
- Python
- Flask
- Postman

## Features
✔ GET all users  
✔ GET a specific user  
✔ POST - Create new user  
✔ PUT - Update user  
✔ DELETE - Remove user  
✔ In-memory database (dictionary)

## How to Run
1. Install dependencies:
   pip install flask
2. Run the app:
   python app.py
3. Open in browser:
   http://127.0.0.1:5000/

## API Endpoints
- GET /users  
- GET /users/<id>  
- POST /users  
- PUT /users/<id>  
- DELETE /users/<id>
